## Code Patterns Discovered

### Pattern: Medical-Safe Derived Computations
```javascript
// App.svelte - TPN calculation with validation
let ingredientsBySection = $derived.by(() => {
  const result = {};
  
  sections.forEach(section => {
    if (section.type === 'dynamic') {
      const keys = extractDirectKeysFromCode(section.content);
      // Medical validation
      const validKeys = keys.filter(key => isValidKey(key) && !isCalculatedValue(key));
      const calculatedKeys = keys.filter(key => isCalculatedValue(key));
      
      result[section.id] = {
        tpnKeys: validKeys,
        calculatedKeys: calculatedKeys,
        customKeys: keys.filter(key => !isValidKey(key)),
        allKeys: keys
      };
    }
  });
  
  return result;
});
```
**When to use**: Complex medical calculations with validation requirements
**Benefits**: Automatic memoization, dependency tracking, type safety
**Medical Safety**: Built-in validation prevents invalid calculations

## Performance Implications

### Positive Optimizations ✅
1. **Reactive Stores**: Automatic dependency tracking reduces unnecessary re-renders
2. **Derived Memoization**: Complex TPN calculations cached until dependencies change
3. **Lazy Loading**: Implemented with `LazyComponent.svelte` for non-critical features
4. **Event Bus**: Decoupled communication prevents prop drilling

### Performance Issues Identified ❌
1. **Monolithic App.svelte**: 1,508 lines cause significant bundle parsing overhead
2. **Complex $derived chains**: Medical calculations triggering frequent updates
3. **Mixed lifecycle patterns**: `onMount` preventing optimization opportunities
4. **Memory pressure**: 40+ state variables in single component

```typescript
// Enhanced medical store pattern
export function createMedicalStore<T>() {
  let state = $state<T>();
  let loading = $state(false);
  let error = $state<Error | null>(null);
  let auditLog = $state<AuditEntry[]>([]);
  
  return {
    get state() { return state; },
    get loading() { return loading; },
    get error() { return error; },
    get auditLog() { return auditLog; },
    
    async updateWithAudit(newState: T, userId: string) {
      // Medical audit trail
      auditLog.push({
        timestamp: new Date(),
        action: 'UPDATE',
        userId,
        oldValue: structuredClone(state),
        newValue: structuredClone(newState)
      });
      state = newState;
    }
  };
}
```
