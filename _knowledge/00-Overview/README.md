# Dynamic Text Editor - Knowledge Base

## Welcome to the Knowledge-First Development Repository

This knowledge base serves as the single source of truth for the Dynamic Text Editor project, a sophisticated medical application for creating and managing TPN (Total Parenteral Nutrition) reference texts with dynamic calculations.

## 🎯 Purpose

This knowledge base enables:
- **Knowledge-First Development**: Make informed decisions based on documented patterns
- **Rapid Onboarding**: New team members can quickly understand the codebase
- **Consistent Development**: Follow established patterns and best practices
- **Performance Optimization**: Apply proven optimization techniques
- **Medical Safety**: Ensure TPN calculation accuracy and patient safety

- Maintain calculation accuracy
- Follow medical data protection standards
- Validate against medical reference ranges
- Support multiple patient populations (neonatal, pediatric, adult)
