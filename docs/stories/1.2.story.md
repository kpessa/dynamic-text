# Story 1.2: Ensure Correct and Secure Dynamic Text Execution

## Status
Complete

## Story
**As a** User,
**I want** The JavaScript I write in dynamic sections to be executed correctly and securely, with full access to the TPN context (`me` object),
**so that** I can create powerful, data-driven reference texts

## Acceptance Criteria
1. When I use `me.getValue('some_key')`, the code must correctly retrieve the value from the active TPN context.
2. When my code has an error, the preview must display a clear error message.
3. All JavaScript execution must happen within the secure Web Worker sandbox.

### Web Worker Sandbox Architecture
**Worker File Location** [Source: public/workers/codeWorker.js]:
- Main worker: `/public/workers/codeWorker.js`
- Uses `importScripts()` to load Babel from CDN
- Creates restricted execution context with limited globals
- Message passing for code execution requests/responses

**Security Boundaries** [Source: architecture/security-and-performance.md, public/workers/codeWorker.js]:
- No DOM access in worker
- No fetch/XMLHttpRequest available
- Limited to Math, Number, String, Array, Object, JSON, Date constructors
- Console.log posts messages back to main thread
- Uses `new Function()` with strict mode

### Babel Configuration
**CDN Loading** [Source: src/lib/utils/lazyBabelCDN.ts]:
```javascript
// Primary CDN with integrity hash
https://unpkg.com/@babel/standalone@7.28.2/babel.min.js
// Fallback CDN
https://cdn.jsdelivr.net/npm/@babel/standalone@7.28.2/babel.min.js
```

**Transpilation Config** [Source: public/workers/codeWorker.js]:
- Preset: 'env'
- Compact output for performance
- Caching of transpiled code (100 item limit)

### TPN Context Structure
**`me` Object Interface** [Source: architecture/data-models.md#tpncontext-model]:
```typescript
interface TPNContext {
  getValue: (key: string) => any;
  ingredients: Ingredient[];
  populationType: 'NEO' | 'CHILD' | 'ADOLESCENT' | 'ADULT';  // Updated types per Story 1.1
  healthSystem: string;
  maxP: (value: number, precision: number) => string;
}
```

**Available Helper Methods** [Source: public/workers/codeWorker.js]:
- `me.getValue(key)` - Access TPN/ingredient values
- `me.maxP(value, precision)` - Precision formatting
- `me.round(value, precision)` - Rounding
- `me.percentage(part, whole)` - Percentage calculations
- `me.convertUnits.mgToG(mg)` - Unit conversions
- `me.medical.bmi(weightKg, heightCm)` - Medical calculations

### Dynamic Content Format
**Syntax Pattern** [Source: architecture/data-models.md]:
```javascript
[f( 
  // JavaScript code here
  var result = me.getValue('keyname');
  return '<html>output</html>';
)]
```
