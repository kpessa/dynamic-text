## Troubleshooting Installation

### Problem: Can't Find Share Button
**Solution**: 
- Ensure you're using Safari, not Chrome or Firefox
- Make sure the page is fully loaded
- Try refreshing the page

### Problem: "Add to Home Screen" Not Visible
**Solution**:
- Scroll down in the share menu - it's usually below the messaging apps
- Make sure you're on the correct website
- Check that you're using Safari browser

### Problem: App Opens in Browser Instead of Full Screen
**Solution**:
- Delete the icon from home screen
- Clear Safari cache: Settings > Safari > Clear History and Website Data
- Reinstall using Method 1

### Problem: App Won't Work Offline
**Solution**:
- Use the app with internet first to cache medical data
- Check Settings > General > iPhone Storage > TPN Editor for storage permission
- Reinstall if offline functionality doesn't activate after first use

### Problem: App Icon Looks Generic
**Solution**:
- This is normal if custom icons aren't configured
- Functionality remains the same
- Contact IT department for custom medical app icons

## Managing the App

### Checking App Version
1. **Open the app**
2. **Look for version number** in settings or about section
3. **Updates happen automatically** when connected to WiFi

### Clearing App Data (If Needed)
⚠️ **Medical Warning**: This will delete all saved calculations

1. **Go to iPhone Settings**
2. **Tap "General" > "iPhone Storage"**
3. **Find "TPN Editor"**
4. **Tap "Offload App"** (keeps data) or "Delete App" (removes all data)
5. **Reinstall** using Method 1 above

### Removing the App
1. **Press and hold** the TPN Editor icon on home screen
2. **Tap "Remove App"**
3. **Choose "Delete App"** to remove completely
4. **Confirm deletion**
